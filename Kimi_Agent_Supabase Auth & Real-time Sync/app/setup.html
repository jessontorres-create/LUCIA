Step 1: Create Supabase Account

First, you need to create a free Supabase account:

  1. Go to supabase.com
  2. Click "Start your project"
  3. Create a new project (e.g., "cc-lucia-kitchen")
  4. Wait for the project to be created
Tip: The free tier includes everything you need for this app!

Step 2: Set Up Database

Run this SQL in your Supabase SQL Editor:

-- Enable extensions CREATE EXTENSION IF NOT EXISTS "uuid-ossp"; -- Create profiles table CREATE TABLE profiles ( id UUID PRIMARY KEY REFERENCES auth.users(id), email TEXT UNIQUE NOT NULL, name TEXT NOT NULL, role TEXT NOT NULL, unit TEXT, created_at TIMESTAMP DEFAULT NOW() ); -- Enable RLS ALTER TABLE profiles ENABLE ROW LEVEL SECURITY;

For the complete schema, see supabase_schema.sql

Then enable Realtime:

  1. Go to Database → Replication
  2. Enable Realtime for all tables
